# ことばモンスターずかん 1ねんせい

小学1年生向けの国語学習ゲームです。
国語の問題を解きながら「ことばモンスター」を集めて育てる、仲間集め型の冒険ゲームです。

- 舞台はオリジナルの世界「ことばランド」
- プレイヤーは「ことばトレーナー」になって、10のステージに挑戦
- 問題に正解すると「ことばパワー」とXPがたまり、仲間のモンスターが成長
- 全ステージをクリアすると「ことばマスター」の称号がもらえます

## 対象学年

小学1年生

## 学習内容

| ステージ | エリア | 内容 |
| --- | --- | --- |
| 1 | はじまりの草原 | ひらがなを読む・見分ける |
| 2 | ことばの森 | 短い言葉と意味の対応 |
| 3 | てんてん山 | 濁音・半濁音 |
| 4 | ちいさい文字のどうくつ | 小さい「ゃ・ゅ・ょ・っ」 |
| 5 | カタカナタウン | カタカナの読み |
| 6 | なかまの広場 | 言葉の仲間分け |
| 7 | ぶんの橋 | 言葉を並べて文を作る |
| 8 | じょしの森 | 助詞「は・を・へ」 |
| 9 | まるてんの海 | 読点「、」と句点「。」 |
| 10 | よみとりの城 | 短い文章の読み取り |

問題は全部で50問以上。すべてローカルで動作し、外部APIは使いません。

## 起動方法

```bash
npm install
npm run dev
```

表示されたURL（通常 http://localhost:5173 ）をブラウザで開いてください。
PC・iPad・スマートフォンに対応したレスポンシブデザインです。

## PWA（ホーム画面インストール・オフライン対応）

このゲームはPWAに対応しています。

- 公開サイト（ https://metaborin.github.io/kotoba-monster-zukan/ ）を開くと、
  ブラウザの「ホーム画面に追加」「インストール」からアプリとして追加できます。
  - iPad / iPhone: Safariの共有ボタン →「ホーム画面に追加」
  - Android: Chromeのメニュー →「アプリをインストール」
  - PC: アドレスバー右端のインストールアイコン
- 一度読み込めば、モンスター画像も含めてキャッシュされるため、
  オフラインでも遊べます（進捗は端末の `localStorage` に保存されます）。
- アプリのアイコンは `public/icons/` にあり、`npm run generate:icons` で
  `public/monsters/aiusagi.png` から再生成できます。

## 問題を追加する方法

[src/data/stages.ts](src/data/stages.ts) を編集します。
各ステージの `questions` 配列に、次の形式で問題を追加してください。

```ts
{
  id: "s1q7",            // ステージ内で重複しないID
  type: "choice",        // "choice"（選択）| "order"（並べ替え）| "fill"（穴埋め）
  question: "「あ」と おなじ もじを えらぼう",
  choices: ["あ", "お", "め", "ぬ"],
  answer: "あ",          // order型のときは ["ねこが", "ねる"] のような配列
  hint: "まるが ちょこんと ついている もじだよ。",
  xp: 10,
}
```

- `fill` 型は問題文に `□` を入れると、その部分が空欄として表示されます。
- `choice`/`fill` の `answer` は `choices` の中の1つと完全一致させてください。
- 読解問題は `storyText` に本文を入れると、問題の上に表示されます。

## モンスターを追加する方法

[src/data/monsters.ts](src/data/monsters.ts) の `MONSTERS` 配列に追加します。

```ts
{
  id: "atarashii-monster",
  name: "あたらしいモンスター",
  icon: "🐣",
  element: "ことば",     // ひらがな | ことば | だくおん | カタカナ | ぶん | よみとり
  description: "せつめい ぶんしょう。",
  skillName: "とくいわざのなまえ",
  skillDescription: "とくいわざの せつめい。",
  unlockStageId: "stage2", // 仲間になるステージのID
}
```

新しいステージの報酬にする場合は、`stages.ts` のステージの
`rewardMonsterId` にモンスターのIDを設定してください。

## 進捗をリセットする方法

次のどちらかで、確認ダイアログのあとに進捗をリセットできます。

- タイトル画面の「はじめから やりなおす」
- せいせき画面（先生・保護者向け）の「はじめからやりなおす」ボタン

進捗はブラウザの `localStorage`（キー: `kotoba-monster-zukan-save-v1`）に
保存されているため、ブラウザのサイトデータ削除でもリセットできます。

## 主なルール

- 正解 +10 XP／ヒントを使って正解 +5 XP／連続正解ボーナス +5 XP／全問正解ボーナス +20 XP
- 4問以上正解でステージクリア（全問正解で金の星、4問で銀の星）
- モンスターのレベル: Lv1=0 / Lv2=30 / Lv3=70 / Lv4=120 / Lv5=200 XP
- なかまチーム（最大3体）に入れたモンスターにXPが入ります
- ステージに合ったモンスターがチームにいると、ヒントを見てもXPが減りません
